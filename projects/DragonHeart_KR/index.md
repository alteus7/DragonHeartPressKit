## Description

드래곤 하트는 같은 색의 블록을 모두 선택하면 없어지는 인디 퍼즐 게임입니다. 여기에 게임에 대한
 간략한 설명을 적습니다.

## History

넥슨, 네오위즈, CCR등의 회사를 거쳐 alteus soft로 독립하여 제작한 첫번째 게임입니다...

## Features

* 새로운 규칙의 블록 퍼즐 게임
* 캐릭터에 따라 게임의 플레이 방식이 달라진다.
* 25종의 캐릭터 지원
* 페이스북, 싱글 모드 지원
